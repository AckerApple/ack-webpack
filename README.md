# ack-webpack
Includes webpack, babel-loader, json-loader, pug-loader and more

## Table of Contents

- Installation
- CLI Commands

## Installation
```
npm install ack-webpack --save-dev
```

## CLI Commands

Example
```
ack-webpack source destination --skip-source-maps --production --minify
```

Options

- **--skip-source-maps**
- **--production**
- **--minify**
